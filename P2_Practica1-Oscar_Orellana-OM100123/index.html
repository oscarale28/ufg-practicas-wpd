¿Qué es el desarrollo web?

El desarrollo web es el proceso de crear y mantener un sitio web funcional en internet, utilizando diferentes lenguajes de programación según el modelo y la parte de la página que corresponda1. Se utilizan tecnologías de software del lado del servidor y del lado del cliente2. En términos funcionales, el desarrollo web se preocupa por integrar las ideas y diseños preestablecidos en un paso anterior
